## Developer notes about AI usage

AI (namely OpenAI's Codex) was used in creating the original code base of this project and writing the code in the src/integration_demo folder.

While AI was also used in some of the documentation, I personally wrote most of it and validated and checked what I wrote with the help of AI. The idea being that if I don't understand the documentation then that means that I don't understand how the code or the system works.

The bulk of the unit and API tests were first written by AI, and then double-checked, and as the case was with some tests, made stronger by me.

The developer is always personally responsible for the code pushed, pulled, and/or shipped.
